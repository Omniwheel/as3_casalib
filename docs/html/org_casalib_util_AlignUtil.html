align

public static function align (
alignment:String, displayObject:DisplayObject, bounds:Rectangle, snapToPixel:Boolean = true, outside:Boolean = false) : void

Aligns a DisplayObject to the bounding Rectangle acording to the defined alignment.
Parameters:
align :
The alignment type/position.
displayObject:
The DisplayObject to align.
bounds :
The area in which to align the DisplayObject.
snapToPixel :
Force the position to whole pixels true, or to let the DisplayObject be positioned on sub-pixels false.
outside :
Align the DisplayObject to the outside of the bounds true, or the inside false.
